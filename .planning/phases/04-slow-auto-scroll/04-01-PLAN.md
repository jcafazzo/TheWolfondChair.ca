**4. Auto-scroll engine (new section after kiosk mode section, before ENGINE section):**

Add a new section `// ═══ AUTO-SCROLL ═══` with these variables and functions:

```javascript
// ═══ AUTO-SCROLL ═══
var autoScrollRAF = null;      // requestAnimationFrame ID
var autoScrollActive = false;  // whether auto-scroll loop is running
var autoScrollStartTime = 0;   // timestamp when scroll began (after initial pause)
var autoScrollPaused = true;   // true during initial 5s pause
var SCROLL_PAUSE_MS = 5000;    // 5-second pause at top before scrolling starts
```

**`startAutoScroll()` function:**
- Called from `enterKiosk()` and from `goToSlide()` (when `isKioskMode` is true).
- First, call `stopAutoScroll()` to clean up any prior animation.
- Get the current slide element: `var slideEl = slides[current];`
- Check if slide has overflow: `slideEl.scrollHeight > slideEl.clientHeight`. If NOT (short slide), call `startKenBurns(slideEl)` and return early.
- If it IS a tall slide: set `slideEl.scrollTop = 0;`, set `autoScrollActive = true`, `autoScrollPaused = true`, `autoScrollStartTime = performance.now()`.
- Start the rAF loop: `autoScrollRAF = requestAnimationFrame(autoScrollTick);`

**`autoScrollTick(timestamp)` function:**
- If `!autoScrollActive` return (guard).
- If `autoScrollPaused`: check if `timestamp - autoScrollStartTime >= SCROLL_PAUSE_MS`. If not, request next frame and return. If yes, set `autoScrollPaused = false`, set `autoScrollStartTime = timestamp` (reset to beginning of actual scroll).
- `var slideEl = slides[current];`
- `var scrollableDistance = slideEl.scrollHeight - slideEl.clientHeight;`
- If `scrollableDistance <= 0`: set `autoScrollActive = false;` and return.
- `var duration = (getSlideDuration() * 1000) - SCROLL_PAUSE_MS - 2000;` — subtract 5s pause at top and 2s pause at bottom before auto-advance (gives viewer time to finish reading the bottom).
- If `duration <= 0`, set `duration = 1000;` (safety floor).
- `var elapsed = timestamp - autoScrollStartTime;`
- `var progress = Math.min(elapsed / duration, 1.0);`
- `slideEl.scrollTop = progress * scrollableDistance;` — linear, teleprompter-style.
- If `progress < 1.0`: `autoScrollRAF = requestAnimationFrame(autoScrollTick);`
- Else: `autoScrollActive = false;` (stay at bottom, wait for auto-advance).

**`stopAutoScroll()` function:**
- If `autoScrollRAF`: `cancelAnimationFrame(autoScrollRAF); autoScrollRAF = null;`
- `autoScrollActive = false;`
- Also call `stopKenBurns();` to clean up any Ken Burns effect.

**`resetSlideScroll(slideEl)` function:**
- `slideEl.scrollTop = 0;`
- Remove any Ken Burns transform: `slideEl.style.transform = ''; slideEl.style.transformOrigin = '';`

**5. Ken Burns effect for short slides:**

**`startKenBurns(slideEl)` function:**
- Called when a slide has no overflow (scrollHeight <= clientHeight).
- Set `autoScrollActive = true;` (reuse active flag for cleanup).
- `autoScrollStartTime = performance.now();`
- `autoScrollPaused = true;` (same 5s pause).
- Start rAF: `autoScrollRAF = requestAnimationFrame(kenBurnsTick);`

**`kenBurnsTick(timestamp)` function:**
- If `!autoScrollActive` return.
- If paused, wait same 5s, then set `autoScrollPaused = false`, set `autoScrollStartTime = timestamp`.
- `var slideEl = slides[current];`
- `var duration = (getSlideDuration() * 1000) - SCROLL_PAUSE_MS;` (use full remaining time for zoom).
- `var elapsed = timestamp - autoScrollStartTime;`
- `var progress = Math.min(elapsed / duration, 1.0);`
- Scale: `var scale = 1 + (0.02 * progress);` — linear from 1.0 to 1.02.
- `slideEl.style.transform = 'scale(' + scale + ')';`
- `slideEl.style.transformOrigin = 'center center';`
- If `progress < 1.0`: `autoScrollRAF = requestAnimationFrame(kenBurnsTick);`
- Else: `autoScrollActive = false;`

**`stopKenBurns()` function:**
- Resets transform on ALL slides (safe cleanup): `for (var i = 0; i < slides.length; i++) { slides[i].style.transform = ''; slides[i].style.transformOrigin = ''; }`
